# MediFlow

Smart Hospital Queue & Appointment Management System — a MERN stack app for managing
hospital appointments and real-time patient queues.

## Tech Stack

- **Frontend:** React (Vite), Tailwind CSS, React Router, Axios
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Auth:** JWT, bcrypt
- **Real-time:** Socket.io (queue updates)

## Project Structure

```
MediFlow/
├── client/   # React frontend
└── server/   # Express backend
```

## Getting Started

### Backend
```
cd server
npm install
npm run dev
```

### Frontend
```
cd client
npm install
npm run dev
```
